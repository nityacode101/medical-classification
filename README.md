
# Medical Image Classification using Deep Learning

This project focuses on detecting diseases like pneumonia using chest X-ray images with the help of deep learning models.

## Tech Stack
 Python
 PyTorch
 OpenCV
 NumPy

## Features
Used transfer learning (ResNet, EfficientNet)
Image preprocessing and augmentation
Model explainability using Grad-CAM

## Challenges
The hardest part was tuning the model to avoid overfitting while maintaining accuracy. Also, making Grad-CAM outputs meaningful took multiple iterations.
